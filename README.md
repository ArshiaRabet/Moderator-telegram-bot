# ربات مدیریت گروه تلگرام

یک ربات پیشرفته برای مدیریت گروه‌های تلگرامی با قابلیت خوشامدگویی، هشداردهی، سکوت موقت و جلوگیری از ارسال لینک توسط کاربران غیرادمین.

## ویژگی‌ها
- خوشامدگویی خودکار به اعضای جدید.
- سیستم هشدار با محدودیت قابل تنظیم و سکوت خودکار پس از رسیدن به سقف هشدار.
- دستورهای مدیریتی:
  - /mute برای سکوت موقت کاربران با تعیین مدت دلخواه.
  - /unmute برای آزاد کردن کاربران میوت‌شده.
  - /lock و /unlock برای قفل یا آزاد کردن چت اعضای گروه.
  - /ban برای حذف و مسدود کردن کاربر.
  - /resetwarns برای پاک کردن هشدارهای کاربر.
  - /warns برای دیدن تعداد هشدار کاربر ریپلای شده.
  - /pin برای پین کردن پیام ریپلای شده.
- محدودسازی ارسال لینک توسط کاربران غیرادمین.
- مشاهده آمار هشدارهای ثبت‌شده با دستور /stats.
- راهنمای کامل با /help یا /komak.

## پیش‌نیازها
- Python 3.10+
- دسترسی به توکن BotFather و دسترسی مدیریتی ربات در گروه.

## نصب و راه‌اندازی
1. مخزن را کلون کنید و وابستگی‌ها را نصب نمایید:
   ```bash
   pip install -r requirements.txt
   ```
2. فایل `.env.example` را به `.env` کپی کنید و مقدار توکن را وارد کنید:
   ```bash
   cp .env.example .env
   ```
3. متغیرهای محیطی را بارگذاری کنید (یا مستقیماً در محیط تنظیم نمایید) و ربات را اجرا کنید:
   ```bash
   export $(grep -v '^#' .env | xargs)
   python main.py
   ```

## متغیرهای محیطی
- `TELEGRAM_BOT_TOKEN` (اجباری): توکن دریافتی از BotFather.
- `WARNINGS_LIMIT` (اختیاری، پیش‌فرض 3): تعداد هشدارهایی که قبل از سکوت خودکار مجاز است.
- `WARNINGS_STORAGE` (اختیاری): مسیر فایل ذخیره هشدارها (پیش‌فرض `./warnings.json`).
- `ADMIN_ONLY_LINKS` (اختیاری، پیش‌فرض true): اگر true باشد، ارسال لینک فقط برای ادمین‌ها مجاز خواهد بود.

## نکات
- برای اعمال هشدار یا سکوت باید پیام کاربر را ریپلای کنید.
- برای عملکرد صحیح، ربات باید در گروه ادمین باشد تا بتواند پیام‌ها را حذف یا کاربران را محدود کند.
